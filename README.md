# Web Scraper Project

## Overview

This is a simple Python Web Scraper that extracts all headings (H1 to H6) from a website using the Requests and BeautifulSoup libraries.

## Features

* Takes a website URL as input
* Sends an HTTP request to the website
* Uses a User-Agent header to avoid access issues
* Extracts all heading tags (H1 to H6)
* Displays the extracted headings

## Technologies Used

* Python
* Requests
* BeautifulSoup (bs4)

## How to Run

1. Install the required libraries:

```bash
pip install requests beautifulsoup4
```

2. Run the Python file:

```bash
python web_scraper.py
```

3. Enter a website URL when prompted.

## Sample URL

```text
https://en.wikipedia.org/wiki/Data_science
```

## Sample Output

```text
Status Code: 200
Data science
Foundations
Etymology
Early usage
```
